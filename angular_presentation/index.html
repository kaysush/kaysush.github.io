What is Angular.js

A JavaScript framework for dynamic web applications.

Why Angular ?

  • Declarative Programming using HTML as templates
  • Dependency Injection
  • Data Binding
  • Reusable Components
  • No more manually registering callbacks
  • No More manual DOM manipulation
  • No More boilerplate code for simple tasks eg. AJAX calls
